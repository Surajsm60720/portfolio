## Running Locally

Clone the repo and install dependencies:

```bash
git clone https://github.com/Surajsm60720/portfolio.git
cd portfolio
npm install
```

Start the development server:

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the site.
